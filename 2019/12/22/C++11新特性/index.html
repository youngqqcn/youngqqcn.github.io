Dec 22, 2019 | C++

C++11/14/17 新特性总结

  • initializer_list
  • auto
  • decltype

用于模板函数的参数返回类型声明

用于对lambda函数类型自动推导

  • override 和 final

override 用于来声明重写父类虚函数
final 用来修饰一个类是,表明一个类禁止被继承; 用来修饰虚函数时, 表明虚函数不能被重写

  • 函数返回类型后置 (用于模板函数)

  • 模板别名: using

using可以用于模板的别名定义, typedef 则不可以

  • nullptr
    用来取代 NULL

  • 智能指针

  • 异常规范
  • explicit

C++11之前仅限制单个参数的构造函数做隐式转换
C+++11开始不限于单个参数的构造函数

  • 可变参数模板
    一般用递归的方式逐步减少参数的个数 , 递归终止于0个参数
  • 右值引用和移动语义

注意自己实现移动拷贝(move ctor)的时候(有指针), 收尾时需要将被move的对象的指针设置为NULL
